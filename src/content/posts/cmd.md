---

### 관리자 명령어

| 명령어    | 사용 예시                         | 설명                                    |
|----------|-----------------------------------|----------------------------------------|
| .ev `code` | .ev chat.reply("응애") | 코드 실행 |
| .ex `code` | .ex aa = "aa" | 코드 실행 |
| !방등록 | !방등록 | 방등록 명령어입니다. |

:::caution
!방등록 명령어를 관리자가 썻는데 한번 더 쓰면 경고 -> 유저 밴 -> 방 밴 순서로 제재가 이루워 질수 있습니다.
:::



:::important
.ev `code` 명령어와 .ex `code` 명령어는 다름.
:::
